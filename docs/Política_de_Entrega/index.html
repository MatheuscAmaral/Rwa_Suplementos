Política de Entrega

Nós temos política de frete grátis para todo o Brasil, com algumas variações de acordo com a região:
• Belo Horizonte e Região: Frete grátis para compras a partir de R$ 99,00
• Região Sul: Frete grátis para compras a partir de R$ 249,00
• Região Centro-Oeste: Frete grátis para compras a partir de R$ 199,00
• Região Nordeste: Frete grátis para compras a partir de R$ 299,00
• Região Norte: Frete grátis para compras a partir de R$ 499,00
Em caso de opção pelo frete via Sedex, o custo passa a ser de responsabilidade do cliente, independente da
região de entrega e valor total do pedido.
A gratificação é concedida automaticamente, sem necessidade de adesão, com base no somatório do valor
dos itens inclusos no pedido e no endereço de entrega informado.
